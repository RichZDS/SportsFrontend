# 前端 (Ant Design + React + ECharts)

## 启动
- Node.js 18+
- 安装依赖：`npm i`
- 开发启动：`npm run dev` (http://localhost:5173)
- 配置后端地址：在 `.env.development` 设置 `VITE_API_BASE=http://localhost:8080`

## 路由
- /admin/customers
- /admin/records
- /admin/reports
- /customer/records

## 说明
- Axios 基于 `VITE_API_BASE`
- 图表组件位于 `src/components/Chart.jsx`
